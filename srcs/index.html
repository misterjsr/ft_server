Ft_server by jeserran

General instructions

  • You have to deliver all the files necessary for your server configuration in a srcs directory.
  • Your Dockerfile will have to be in the root of your repository. It will be used to build your container. You cannot use docker-compose.
  • All the files necessary for your wordpress will have to be in the srcs directory.

Mandatory part

  • You will have to install a web server with Nginx, inside a single Docker container. The container will have to run on Debian Buster.
  • Your server will have to allow multiple services to run at the same time. The services will be a Wordpress, which you will have to install yourself, Phpmyadmin and MySQL. You will have to make sure that your SQL database works with wordpress and with phpmyadmin.
  • When possible, your server should be able to use the SSL protocol.
  • You will have to ensure that your server redirects to the correct website, based on the url entered. You also need to ensure that your server runs with an automatic index, which can be disabled.
